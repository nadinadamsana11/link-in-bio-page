# ⚡ Link-in-Bio Platform

A premium, production-ready Link-in-Bio platform designed for digital creators who demand speed, style, and professional identity management.

## 🚀 Key Features

### 🪪 Identity Badge (NIC Style)
A high-end, badge-inspired profile card that serves as your digital identity. Features include a verified badge, custom bio, and a professional layout that looks great on any device.

### 🖼️ Cloudinary & WebP Optimization
- **Smart Media Management**: All media is uploaded directly to **Cloudinary** for lightning-fast delivery.
- **Client-Side Optimization**: Images are automatically processed and resized on the client side before upload to ensure minimal file size without compromising professional quality.

### 🔗 Interactive Link Library
- **Card-Based UI**: Links are presented in a sleek grid of interactive cards.
- **Dynamic Animations**: Smooth hover-to-scale effects and background transitions create a premium user experience.
- **Custom Icons**: Support for Lucide icons to represent your various digital touchpoints.

### 🔐 Secure Identity Management
- **Unique Usernames**: A global uniqueness check ensures no two creators share the same handle.
- **Flexible Identity**: Change your public username and display details at any time with real-time updates.

### 🌓 Clean Slate Design
- Built with a specialized **Slate-Gray palette** for a modern, high-contrast aesthetic.
- Fully responsive layout ensuring consistent branding across mobile and desktop.

## 🛠️ Tech Stack

- **Core**: Vanilla JavaScript (ES6+ Modules)
- **Styling**: Tailwind CSS & Vanilla CSS
- **Backend**: Firebase (Auth, Firestore)
- **Media**: Cloudinary (Optimized Image Delivery)
- **Icons**: Lucide Icons

## 📦 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/nadinadamsana11/link-in-bio-page.git
    ```
2.  **Firebase Config**:
    Update `assets/js/firebase-config.js` with your own project credentials.
3.  **Deployment**:
    Push to any static host (GitHub Pages, Vercel, Netlify) and you're ready to go!
